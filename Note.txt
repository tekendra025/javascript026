######Types of Variable
            1.Var:
             - It is a old method to declare varible.
             -It can be redeclare:
                          var age = 50
                           var age = 12
                            console.log(age) //output 12
             - It can be update:
                          var age = 50
                             age = 12 //only update no use var
                            console.log(age) //output 12
            - It is global scope:
                         -means if it is declare ouside the function or inside {}-(block scope) it can be access everywhere in program 
                          -but it doesnt means if i declare var inside fucntion then it cannot be declare ouside the function. This is called local scocpe
                          
                          var a = 20
                          function avg(){
                            var b = 30
                            console.log(a) //it is true but
                           }
                           console.log(b) //here b cannot be access because b is a local scope. 
                          

            2.let:
              - It is latest method
              -It cannot be redeclare:
                       let city = "ktm"
                       let city = "Pokhara"
                        console.log(city) //it prints error
            -It value can be change:
                       let city = "ktm"
                        city = "Pokhara"
                        console.log(city) //it prints pokhara
             - it is block scope: 
                     block scope means variable declare inside {}. Function also works . If variable inside fucntion then it is called local and if inside {} it is block scope. It works in both           
                     
                     if (true) {
                     let a = 10; //a is a block scope inside {}
                      console.log(a); // ✅ works
                              }

                       console.log(a); // ❌ error

                 - also works inside function 
                  
                   function test() {
                     let a = 10; //here a is a local scope
                    console.log(a); // ✅ works
                     }
                     console.log(a); // ❌ error
            
            Block scope ≠ Local scope
            Block scope is a type of local scope
            
            
            
            3.const:
                -its value is constant/fixed through the program
                -Its value cannot be redeclare
                -Its value cannot be change/update
                     const a = 10
                     const  a = 20; or simpley a = 20 -no update value
                     console.log(a) //error
                
                -It is alos a block scope like let
